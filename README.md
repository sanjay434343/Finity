1. **Clone the repository**
   ```bash

   cd Finity
   ```

2. **Install dependencies**

3. **Configure Firebase** (Optional)
   ```bash
   # Add your google-services.json for Android
   # Add your GoogleService-Info.plist for iOS
   ```


```bash

flutter build ios --release
```
